# WEKA UI Components

## Storybook
https://weka.github.io/weka-ui-components

## Available Scripts

### Build the library

```
yarn build
```

### Publish the library

```
yarn publish
```

### Run dev rollup

```
yarn dev
```

### Run storybook locally

```
yarn run storybook
```

### Build storybook

```
yarn run build-storybook
```

### Deploy storybook

```
yarn run deploy-storybook
```
